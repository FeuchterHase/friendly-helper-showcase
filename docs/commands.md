# Commands

Diese Datei zeigt beispielhaft, welche Art von Slash-Commands Friendly-Helpers bereitstellt.

Die konkrete Command-Struktur kann sich waehrend der aktiven Entwicklung aendern.

## Schiff und Schiffsdaten

| Command | Beschreibung |
| --- | --- |
| `/info` | Zeigt Informationen zu einem Schiff |
| `/vergleich` | Vergleicht zwei Schiffe |
| `/liste` | Listet Schiffe nach Filtern |
| `/zufallsschiff` | Gibt ein zufaelliges Schiff aus |
| `/schiffstats` | Zeigt Spielerwerte auf einem bestimmten Schiff |
| `/schiffduell` | Vergleicht zwei Spieler auf demselben Schiff |

## Spieler

| Command | Beschreibung |
| --- | --- |
| `/mystats` | Zeigt Spielerstatistiken nach Modus |
| `/meineschiffe` | Zeigt gespielte Schiffe eines Spielers |
| `/spielerduell` | Vergleicht zwei Spieler |
| `/rekorde` | Zeigt Spielerrekorde |
| `/errungenschaften` | Zeigt Errungenschafts-Zusammenfassung |
| `/expert` | Zeigt Abzeichen eines Spielers |
| `/top_schaden` | Zeigt hoechste Schadenswerte je Schiff |

## Clan

| Command | Beschreibung |
| --- | --- |
| `/clanmitglieder` | Listet Clanmitglieder |
| `/leaderboard` | Erstellt Clan-Leaderboards |
| `/clanrang` | Zeigt Spielerposition im Clan |
| `/clanrekorde` | Zeigt Clanrekorde |
| `/clanwars` | Zeigt Clan-Wars-Auswertung |
| `/clan-rankings` | Zeigt Clan-Rankings |

## Web-Rankings

| Command | Beschreibung |
| --- | --- |
| `/spieler-rankings` | Zeigt Spieler-Ranking-Auszug |
| `/schiffs-meta` | Zeigt serverweite Schiffs-Meta |

## Server- und Clan-Stats

| Command | Beschreibung |
| --- | --- |
| `/serverstats` | Verwaltet Server-Stats-Channels |
| `/clanserver` | Verknuepft einen Clan mit dem Discord-Server |
| `/clanserverstats` | Verwaltet Clan-Stats-Channels |
| `/notifications` | Verwaltet Benachrichtigungsfunktionen |

## Allgemein

| Command | Beschreibung |
| --- | --- |
| `/help` | Zeigt eine private Befehlsuebersicht |
| `/aboutme` | Erklaert den Bot und die Projektidee |

## Weblinks

Viele Commands verlinken auf passende Seiten von:

So bleibt Discord kompakt, waehrend Detaildaten in der Webansicht verfuegbar sind.
